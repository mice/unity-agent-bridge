# AGB_186 - Legacy settings default MCP plugin migration

## Scope

Proves that an existing Agent Bridge settings asset missing newer default plugin registrations is migrated in memory to the complete current built-in plugin set.

## Setup

- Start with the current default settings.
- Retain only the legacy MonoBehaviour semantics and Lua tools registrations.
- Invoke the default-registration migration path.

## Assertions

- ProjectInfo, EditorBasics, UnityQueries, TestRunner, and RoslynExecution registrations are restored.
- Existing MonoBehaviour semantics and Lua tools registrations remain present.

## Risk Covered

An external Unity project with a legacy settings asset could otherwise start a healthy bridge that lacks the frozen MCP tool providers, producing MCP009/MCP010 false failures.
